🔧 Instrucciones para Pruebas Completas:

# 1. Usar Extensiones del Navegador: - Chrome: "Location Guard" o "User-Agent Switcher" - Firefox: "User-Agent Switcher and Manager" # 2. Usar VPN (Recomendado): - NordVPN, ExpressVPN, o VPN gratuita - Conectarse a diferentes países y probar # 3. Usar DevTools del Navegador: - F12 → Console → ejecutar comandos de prueba - Network → modificar headers # 4. Usar Herramientas Online: - whatismyipaddress.com para verificar IP - browserleaks.com para verificar geolocalización
