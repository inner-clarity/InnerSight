# InnerSight

# すべての神の子の包含のための情報に基づいた支援

福音の文脈で精神的健康と神経学的ニーズをより良く理解するための教育リソース

# 目的

InnerSightは、末日聖徒イエス・キリスト教会の指導者、家族、会員が、神経学的な違いやトラウマ反応が正当な形の感情調節を必要とする場合があることをより良く理解するための支援を求める教育リソースです。この作品は臨床的証拠、福音の原則、そしてキリストのような愛で奉仕したいという共通の願望に基づいています。

# 基盤

このリソースは、精神的健康や神経学的ニーズが特定の適応を必要とする忠実な会員をどのように支援するかについて、指導者や家族からよくある質問に答えます。認識された臨床基準（DSM-5など）、神経科学研究、そして回復された福音の原則に基づいています。

# 証言

思慮深い兄弟:
"私はメルキゼデク聖職の保持者で、複数の召しで奉仕してきました。私の神経学的状態は感覚調節のための特定の適応を必要とします。理解のあるビショップが私に言うまで、何年も恥を感じていました：'神の前でのあなたの尊厳は、あなたの神経多様性に依存しません'。その言葉が私の人生を変えました。"

忠実な会員:
"指導者との面談で、個人ケア用品（PEPT）の使用の必要性を提起したとき、この指導者は正当性を疑問視し、それが'価値ある聖職保持者がそれを使用することは矛盾している、または悪く見える、または恥ずかしい'と明確にしました。それが気まぐれではなく神経学的ニーズであることを理解せずに。誤解されていると感じましたが、神が私に何が起こっているかを知っておられ、私の心を理解しておられることを感じました。"

帰還宣教師:
"私の宣教師期間中、他の人々を不快にさせる状況にさらされました。時々チックがあり、無意識の言語音（トゥレット症候群）を発することさえあります。私を'不敬虔または不信心'として注意する指導者もいましたが、私は無意識にそれをしていました。彼らがこれらの神経学的状態について情報を持っていればよかったと思います。愛深く理解のある宣教師会長を持てたことに感謝し、非常に忍耐強い同伴者を持ちました。彼らは私に救い主が私を世話してくださっていることを感じさせてくれました。私は名誉をもって宣教師を終えました"

自己犠牲的な母親:
"多くの人が私に'なぜ息子を静かにさせなかったのか'や'私は悪い母親だ'と言いました。息子をコントロールしないからです（息子は自閉症です）。彼らは教会に行くのをやめる可能性さえ提案しました。彼らは私が毎日経験するすべてを理解していません。それでも、私を愛してくださる天の父から、愛深く忠実で善良な夫である父から、そして私が忠実であり続けるなら'主が私を肩に担いでくださる'という約束から力を得ています。他の人がより大きな理解を持つことができれば、私の重荷を少し軽く感じるでしょう。しかし何よりも、救い主が私を完全に理解してくださることを知っており、それが私に平安を与えます。"

# 主要内容

# 心理学的基盤

このセクションは、特にトラウマを経験した人や神経学的な違いがある人がどのように感情を調節するかについての臨床的観点を提示します。現在の医学的基準、神経科学、国際診断ガイドラインを含めて、特定の反応が障害ではなく必要な適応戦略である場合を説明します。

# 感情調節ニーズの臨床的理解

特定の行動が個人的な好みではなく神経学的ニーズやトラウマ反応を反映する時期を認識するための証拠に基づくガイド。臨床基準、例示的な例、理解を持って奉仕したい指導者や家族のための指導を含みます。

# 必要な調節戦略の放棄を要求することの影響

このセクションは、神経科学と臨床研究に基づいて、誰かに本質的な感情調節戦略の使用を停止するよう求めること（良い意図でも）が追加の苦痛、機能的悪化、霊的距離感を引き起こす方法を説明します。

# 愛が同調に条件付けられるとき

一部の会員は、自分の所属や霊的価値が特定の感情調節ニーズを放棄することに依存していると感じていると報告しています。このセクションは、これらの力学が（意図しないものでも）信仰、自尊心、関係にどのように影響するかを分析し、条件を課すことなく奉仕する原則を提供します。

# 感情調節の本質

一部の人々は特定の感情調節戦略をアイデンティティや性的嗜好の問題と混同します。このセクションは、これらのニーズが性的指向ではなく精神的・神経学的健康に関連する理由について、神経科学に基づく明確な臨床的説明を提供します。

# 感情的婚姻安全

このセクションは、婚姻の親密さにおける強制的な制限が配偶者の感情的安心と精神的健康にどのように影響するかを検討します。正当な感情調節を妨げる制限が課されたときの婚姻関係への影響を分析し、福音の原則と婚姻親密さの臨床的理解に基づく観点を提供します。

# 霊的尊厳と精神的健康

このセクションは、福音の教え—聖典、家族宣言、現代の啓示を含む—が会員の神の前での尊厳がその人の精神的健康や神経学的状態ではなく、その人の信仰、悔い改め、キリストに従いたいという願望に依存することをどのように確認するかを検討します。

# 制度的提案

精神的健康と神経学的状態における制度的包含のための大管長会に提出された包括的な提案。科学的基盤、教義的調和、指導者とコミュニティのための明確な指導を含む提案された制度的声明を含みます。

# 指導者のための実用的ガイド

感情調節ニーズを持つ会員に奉仕する際に指導者が直面する一般的なシナリオを含む実用的ガイド。実際の状況で知恵と慈悲をもって応答する方法についての具体的な指導を含みます。

# 指導者のためのよくある質問

精神的健康と神経学的ニーズを持つ会員を支援する方法をより良く理解しようとする指導者のための一般的な質問と回答。正当なニーズの区別、神殿の価値、家族力学、合理的配慮についての実用的指導を含みます。

# 困難な家族会話のためのガイド

精神的健康や神経学的ニーズについての挑戦的な会話に直面する会員や家族のための実用的ガイド。ニーズを持つ人と家族構成員の両方のための具体的な指導を含み、コミュニケーション戦略、感情管理、専門的支援の探求を含みます。

# 目標

以下を通じて指導者、家族、会員を支援：

  • 精神的健康と神経学的ニーズのより良い理解
  • より大きな慈悲と知識で奉仕
  • 個人の状況に対する感受性を持って福音の原則を適用
  • より大きな理解と相互支援を通じてコミュニティを強化
